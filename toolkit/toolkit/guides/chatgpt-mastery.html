🎯

Be Specific

Instead of "Help me write," try "Help me write a professional email declining a job offer while maintaining a positive relationship."

📋

Provide Context

Share relevant background information. "I'm a marketing manager at a tech startup" gives better context than no details.

🔄

Use Examples

Show the format you want: "Write like this example..." or "Use this tone..." to get consistent results.

🎭

Assign Roles

"Act as a financial advisor," "You're a creative director," or "Be my coding mentor" for specialized responses.

📐

Set Constraints

Specify length, format, audience: "Write a 100-word summary for executives" vs. "Explain this concept."

🔁

Iterate & Refine

Build on responses: "Make it more formal," "Add specific examples," "Focus on the technical aspects."

❌ Poor Prompt

"Help me with marketing"

Problems: Too vague, no context, unclear goals

✅ Excellent Prompt

"I'm launching a fitness app for busy professionals aged 25-40. Help me create 5 social media campaign ideas that emphasize convenience and quick results. Each idea should include a hook, key message, and suggested visual concept. Keep the tone motivational but realistic."

Why it works: Specific context, clear audience, defined deliverable, format specified, tone guidance

❌ Poor Prompt

"Write code"

Problems: No language specified, no functionality described, no context

✅ Excellent Prompt

"Write a Python function that takes a list of email addresses and validates them using regex. Include error handling for invalid formats and return both valid emails and a count of invalid ones. Add comments explaining the regex pattern and include 2-3 test cases."

Why it works: Language specified, clear functionality, requirements detailed, examples requested

🚫 AI gives generic responses

Solution: Add specific context, constraints, and examples

Try: "Act as [specific role] helping [specific audience] with [specific goal]. Use [tone/style]. Format as [structure]."

🔄 Responses are inconsistent

Solution: Provide clear formatting examples and style guidelines

Try: "Follow this exact format: [example]. Keep the same tone throughout."

❌ AI refuses reasonable requests

Solution: Clarify your intent and provide educational context

Try: "I'm learning about [topic] for [legitimate purpose]. Please help me understand [specific aspect]."

📊 Need more detailed analysis

Solution: Request step-by-step reasoning and multiple perspectives

Try: "Think through this step by step. Consider pros/cons and provide reasoning for your recommendations."
